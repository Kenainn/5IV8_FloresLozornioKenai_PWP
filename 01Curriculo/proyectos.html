Mis Proyectos

He participado en proyectos escolares que me han permitido aplicar conocimientos teóricos a soluciones reales:

GreenPoint (4º semestre): Aplicación en Python con Tkinter y ttkbootstrap como interfaz, conectada a MySQL Server, diseñada para facilitar el tradeo de bolsas de reciclaje en tiendas de conveniencia. El sistema permite que los usuarios obtengan incentivos para la canasta básica, mientras que las tiendas apoyan a los centros de reciclaje o gestionan los pagos correspondientes, promoviendo la eficiencia y la responsabilidad ambiental. Este proyecto reforzó mis habilidades en gestión de bases de datos y lógica de negocio, aspectos clave del backend.

CRECIVE (3º semestre): Aplicación en Python con Tkinter que calcula conversiones de aceites a jabones, velas y esencias para empresas con maquinaria especializada. Incluye un gestor de usuarios implementado con CSV, desarrollado antes de cursar materias de bases de datos, demostrando iniciativa y capacidad para manejar información de manera práctica y organizada.
